# Gestão de Rotas

API REST NestJS para **gestão de frota** (Aivacol) — marcas, modelos, veículos, JWT, Redis, Prometheus/Grafana.

## Documentação

**https://vitorjobs.github.io/infosistema-gestao-frotas-lab/**

## Tecnologias

NestJS · TypeORM · SQL Server · JWT · Redis · MongoDB · Prometheus · Grafana · VitePress

## Links

| Recurso | URL |
|---|---|
| Swagger | http://localhost:3001/api/docs |
| Grafana | http://localhost:3002 |
| Prometheus | http://localhost:9090 |
| Repositório | https://github.com/vitorjobs/infosistema-gestao-frotas-lab |

## Início rápido

```bash
cp .env.example .env
docker compose up -d --build
```

Execução completa, variáveis e testes: [documentação VitePress](https://vitorjobs.github.io/infosistema-gestao-frotas-lab/).
